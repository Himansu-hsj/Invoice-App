# Invoice-App

Full Stack Invoice Application built with:

- React (Vite)
- Node.js (Express)
- MySQL
- JWT Authentication

## Features
- Customers management
- Invoices (Draft / Sent / Paid)
- Partial & Full Payments
- PDF Invoice & Payment Receipt
- Authentication (Login / Logout)

## Tech Stack
Frontend: React + Vite  
Backend: Node.js + Express  
Database: MySQL
